# 📊 帳務資料自動化整合工具 (Data Integration Tool)

這是一個基於 Streamlit 開發的數據處理工具，旨在協助內部團隊自動化整合每日數據與各類彙總模板。

## 🌟 主要功能

- **自動代碼映射**：自動比對資料來源與主模板，確保資料歸戶正確。
- **多來源表處理**：支援整合多種不同性質的來源工作表，實現單一界面處理。
- **複雜邏輯自動化**：內建數據核對與加總邏輯，取代人工手動填寫工作。
- **彈性模板支援**：支援指定格式的 Excel 年度報表模板，確保資料填入正確位置。

## 🛠️ 使用說明

1. **準備檔案**：請確保您的彙總模板檔案與每日數據來源檔案格式正確。
2. **上傳檔案**：在操作介面上傳對應的 Excel 檔案。
3. **執行處理**：點選處理按鈕，系統將自動進行邏輯核對與資料填入。
4. **下載結果**：處理完成後即可下載最新的彙總檔案。

## 💻 開發與運行

本專案使用 Python 環境開發。

### 必要步驟
1. 安裝環境依賴：`pip install -r requirements.txt`
2. 啟動應用程式：`streamlit run app.py`

## 📦 技術元件
- `streamlit`: 視覺化介面
- `pandas`: 資料處理核心
- `openpyxl`: Excel 處理插件
- `numpy`: 計算模組
